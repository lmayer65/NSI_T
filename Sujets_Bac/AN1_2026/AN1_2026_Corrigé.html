7. Méthode colonne_score_min
def colonne_score_min(self):
    meilleur = self.suivants[0]

    for noeud in self.suivants:
        if noeud.score < meilleur.score:
            meilleur = noeud

    return (meilleur.colonne, meilleur.score)

def choisit_coup(grille, joueur):

    for colonne in range(7):

        if grille2.joue(colonne, joueur):

            racine.suivants.append(noeud)

def est_strict_croissante(seq):
    for i in range(1, len(seq)):
        if seq[i] <= seq[i - 1]:
            return False
    return True
11. Fonction récursive llsc_fin
def llsc_fin(tab, i):
    if i == 0:
        return 1

    max_len = 1

    for j in range(i):
        if tab[j] < tab[i]:
            max_len = max(max_len, llsc_fin(tab, j) + 1)

    return max_len
12. Fonction dynamique llsc_dyn
def llsc_dyn(tab):
    n = len(tab)
    dyn = [1] * n

    for i in range(1, n):
        for j in range(i):
            if tab[j] < tab[i]:
                dyn[i] = max(dyn[i], dyn[j] + 1)

    return max(dyn)
13. Avantage de la programmation dynamique

L’implémentation dynamique évite de recalculer plusieurs fois les mêmes valeurs.

C’est une programmation dynamique ascendante : on calcule d’abord les solutions des sous-problèmes les plus simples, puis on les utilise pour construire les solutions suivantes.

Ici, dyn[i] est calculé à partir des valeurs déjà connues dyn[j] pour j < i.
