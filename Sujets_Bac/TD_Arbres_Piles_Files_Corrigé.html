    lst.append(proc)

3. Contenu final des piles

Après exécution de la séquence :

enfiler(file, ps6)
defiler(file)
defiler(file)
defiler(file)
enfiler(file, ps7)

on obtient :

  • p1 contient uniquement ps7 ;
  • p2 contient (du bas vers le haut) ps6, ps5, ps4.

4. Fonctions sur une file implémentée par deux piles

4.a est_vide(f)

Une file est vide si et seulement si les deux piles sont vides.

def est_vide(f):
    """Renvoie True si la file f est vide, False sinon."""
    return pile_vide(f[0]) and pile_vide(f[1])

4.b enfiler(f, elt)

Enfiler revient à empiler dans p1.

def enfiler(f, elt):
    """Ajoute elt en queue de la file f."""
    empiler(f[0], elt)

4.c defiler(f)

Si p2 est vide, on transfère tout p1 vers p2 pour inverser l’ordre, puis on dépile p2.

def defiler(f):
    """Retire et renvoie l'élément en tête de la file f."""
    p1, p2 = f[0], f[1]

    if pile_vide(p2):
        while not pile_vide(p1):
            empiler(p2, depiler(p1))

    return depiler(p2)
