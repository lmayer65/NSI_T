Exercice 2 – Piles et files

Une méthode simple pour gérer l’ordonnancement des processus est d’exécuter les processus une seule fois et dans leur ordre d’arrivée.

1. Structure FIFO

Parmi les propositions suivantes, quelle est la structure de données la plus appropriée pour mettre en œuvre le mode FIFO (First In First Out) ?

a) liste
b) dictionnaire
c) pile
d) file

2. Implémentations de file

On choisit de stocker les données des processus en attente à l’aide d’une liste Python lst.

On dispose déjà d’une fonction retirer(lst) qui renvoie l’élément lst[0] puis le supprime de la liste lst. Écrire en Python le code d’une fonction ajouter(lst, proc) qui ajoute à la fin de la liste lst le nouveau processus en attente proc.

On choisit maintenant d’implémenter une file file à l’aide d’un couple (p1, p2) où p1 et p2 sont des piles. Ainsi file[0] et file[1] sont respectivement les piles p1 et p2.

  • Pour enfiler un nouvel élément elt dans file, on l’empile dans p1.
  • Pour défiler file, deux cas se présentent :
    • Cas 1 : la pile p2 n’est pas vide → on dépile p2.
    • Cas 2 : la pile p2 est vide → on dépile les éléments de p1 en les empilant dans p2 jusqu’à ce que p1 soit vide, puis on dépile p2.

4. Fonctions à écrire

On dispose des fonctions suivantes :

  • empiler(p, elt) : empile l’élément elt dans la pile p ;
  • depiler(p) : renvoie le sommet de la pile p si elle n’est pas vide et le supprime ;
  • pile_vide(p) : renvoie True si la pile p est vide, False sinon.

On note f une file représentée par le couple de piles.

a)

b)

Écrire en Python une fonction enfiler(f, elt) qui ajoute elt en queue de la file représentée par f.

c)

Écrire en Python une fonction defiler(f) qui renvoie l’élément en tête de la file représentée par f en le retirant.
